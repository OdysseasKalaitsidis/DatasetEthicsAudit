Τίτλος Εργασίας: Διαδραστικό Εργαλείο Ελέγχου Αλγοριθμικής Ηθικής σε Δεδομένα (Data Ethics Analyzer)
Deadline: 31/1/2026, ΩΡΑ: 23:55
Περιγραφή
Θα αναπτύξετε μια διαδραστική εφαρμογή web (R Shiny ή Python Streamlit) που θα λειτουργεί ως εργαλείο ελέγχου και αξιολόγησης ηθικών ζητημάτων σε tabular datasets. Η εφαρμογή θα επιτρέπει στους χρήστες να φορτώσουν ένα dataset και θα παράγει αυτόματα αναφορές σχετικά με πιθανά ηθικά προβλήματα.
Απαιτούμενες Λειτουργίες
1. Φόρτωση Δεδομένων: Η εφαρμογή θα πρέπει να επιτρέπει τη φόρτωση δεδομένων σε μορφή CSV ή Excel και να παρέχει βασική προεπισκόπηση των δεδομένων (πρώτες γραμμές, τύποι μεταβλητών).
2. Ανίχνευση Bias & Fairness Issues 
Η εφαρμογή σας θα πρέπει να υλοποιεί 2-3 διαδικασίες ελέγχου από τις παρακάτω (ή άλλες δικές σας):
Ενδεικτικές ιδέες:
Έλεγχος για ανισότητες σε ευαίσθητα χαρακτηριστικά (φύλο, ηλικία, εθνικότητα) (Demographic Bias Detection)
Εντοπισμός ανισορροπίας στις κατηγορίες (target variable) (Class Imbalance)
Υποεκπροσώπηση συγκεκριμένων ομάδων (Representation Issues)
Συσχετίσεις features με προστατευμένα χαρακτηριστικά (Correlation with Protected Attributes)
3. Αλγοριθμικοί Έλεγχοι 
Η εφαρμογή σας θα πρέπει να υλοποιεί 2-3 διαδικασίες ελέγχου από τις παρακάτω (ή άλλες δικές σας):
Ενδεικτικές ιδέες:
Προειδοποιήσεις για πιθανό overfitting (πολλά features, λίγα δεδομένα) (Overfitting Indicators)
Έλεγχος για features που μπορεί να προκαλούν leakage (Data Leakage Detection)
Ανάλυση missing values και πιθανές συνέπειες (Missing Data Patterns)
Επίδραση outliers σε αλγοριθμικές αποφάσεις (Outliers Impact)
4. Προαιρετικά (Bonus)
Fairness Metrics (Demographic Parity, Equal Opportunity)
Visualization των bias patterns
Απόδοση ML διεργασιών
Σημαντικό: 
Ο στόχος δεν είναι να μάθετε προγραμματισμό ή machine learning, αλλά να κατανοήσετε και να εντοπίσετε ηθικά ζητήματα στα δεδομένα και τους αλγορίθμους.
Επιτρέπεται η χρήση εργαλείων τεχνητής νοημοσύνης (AI), όπως Large Language Models (LLMs), GPT και άλλα generative AI tools, για την εκπόνηση της εργασίας.
